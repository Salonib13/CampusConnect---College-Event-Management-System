Campus Connect – College Event Management System

Project Overview : 

Campus Connect is a role-based college event management system designed to connect students and administrators. Students can browse and register for events, while admins can create, manage, and monitor events.
This project demonstrates my experience in full-stack web development, secure authentication, and database management, integrating frontend and backend functionalities into a seamless workflow.

Features :

1) Student Module
- Role-based registration and login
- Passwords securely hashed using bcrypt
- Dashboard to view upcoming events and register with a single click
- Track registration status for each event
2) Admin Module
- Secure login for admins (predefined credentials)
- Create and post new events
- Open or close event registration
- Monitor student registrations

Technology Stack :

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js
- Database: MySQL
- Security: bcrypt

Project Workflow :

1) Role Selection: User chooses Student or Admin on the landing page
2) Student Flow: Registration → Login → Dashboard → View & Register Events
3) Admin Flow: Login → Dashboard → Create & Manage Events → Monitor Registrations

Security Implementation :

- Passwords are securely hashed using bcrypt before storing in the database
- Role-based access control ensures that only authorized users can perform specific actions

Skills & Learning

- Developed dynamic dashboards for students and admins
- Implemented secure authentication and role-based access
- Created a full-stack application integrating frontend and backend
- Learned to design logical workflows and manage event registration data efficiently
